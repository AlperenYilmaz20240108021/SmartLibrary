SmartLibrary Projesi

Bu projeyi Java ve SQLite kullanarak gelistirdim. Basit bir kutuphane yonetim uygulamasidir.

Neler yapiyor:
- Kitap ve ogrenci ekleyebiliyoruz.
- Kitaplari odunc verip geri alabiliyoruz.
- Veritabani baglantisi otomatik yapiliyor.

Kullanilan teknolojiler:
Java, JDBC, SQLite ve Maven.

Nasil calisir:
Projeyi Intellij IDEA ile acip Main.java dosyasini calistirmaniz yeterli. Veritabani dosyasi (library.db) otomatik olusacaktir.
